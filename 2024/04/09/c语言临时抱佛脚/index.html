字符串

为了避免读入字符串越界

  1. strlen：

    • 功能：计算给定字符串的长度，不包括结尾的空字符\0。

    • 用法：size_t strlen(const char *str);

    • 示例：

  2. strcmp：

    • 功能：比较两个字符串是否相等。

    • 用法：int strcmp(const char *str1, const char *str2);

    • 返回值：如果 str1 和 str2 字符串相同，则返回 0。如果 str1 小于 str2，返回小于0的值。如果 str1 大于 str2，返回大于0的值。

    • 示例：

  3. strcpy：

    • 功能：复制字符串，包括结尾的空字符\0。

    • 用法：char *strcpy(char *dest, const char *src);

    • 示例：

  4. strcat：

    • 功能：将一个字符串附加到另一个字符串的末尾。

    • 用法：char *strcat(char *dest, const char *src);

    • 示例：

  5. strchr：

    • 功能：在字符串中查找第一次出现的指定字符。

    • 用法：char *strchr(const char *str, int c);

    • 返回值：返回指向字符串中第一次出现字符c的指针。如果未找到该字符，则返回NULL。

    • 示例：

  6. strstr：

    • 功能：在一个字符串中查找第一次出现的另一个字符串。

    • 用法：char *strstr(const char *haystack, const char *needle);

    • 返回值：如果找到了needle字符串，则返回第一次出现的位置的指针。如果未找到，则返回NULL。

    • 示例：
