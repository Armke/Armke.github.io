The isalpha function is a standard C++ function used to check if a given character is an alphabetic letter (either uppercase or lowercase).

substr

在 C++ 中，std::string 类的 substr 方法用于从字符串中提取一个子字符串。该方法的原型如下：

  • pos 参数指定子字符串开始的位置（以 0 为起始索引）。
  • len 参数指定要提取的子字符串的长度。如果 len 超过了从 pos 开始到字符串末尾的长度，则会提取从 pos 开始到字符串末尾的所有字符。
  • 如果不提供 len 参数，substr 将返回从 pos 开始到原字符串末尾的子字符串。

printf格式化

整数格式化

  • %d 或 %i：按照十进制格式输出整数。
  • %u：无符号十进制整数。
  • %o：无符号八进制数。
  • %x 或 %X：无符号十六进制数，%x输出小写字母，%X输出大写字母。
  • %02d：输出至少两位的十进制数，不足部分前面补充0。

浮点数格式化

  • %f：浮点数输出。默认情况下，printf会根据需要输出6位小数。
  • %.2f：输出浮点数，但是只保留小数点后两位。
  • %e 或 %E：使用科学计数法格式输出浮点数，%e输出小写e，%E输出大写E。
  • %g 或 %G：自动选择%f或%e（%E）中较短的一种输出，且不会输出无意义的零。

字符和字符串格式化

  • %c：输出单个字符。
  • %s：输出字符串。
  • %.5s：输出字符串的前5个字符。

宽度和精度

  • %5d：输出宽度为5的整数，右对齐，不足部分以空格填充。
  • %-5d：输出宽度为5的整数，左对齐，不足部分以空格填充。
  • %8.2f：输出宽度为8的浮点数，其中小数点后有两位数字，不足部分以空格填充。

特殊字符

  • %%：输出百分号本身。
