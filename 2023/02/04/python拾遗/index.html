python拾遗

print

print()在打印之后会自动加一个换行，所以在打印一些自带换行的内容时候会导致了两个换行

可以使用rstrip(),strip()去除，

其中rstrip()是去除右边的空白,strip()是去除左右的空白，还有lstrip()去除左侧的

列表

a为列表

  • 追加a.append()
  • 插入a.insert()
  • 删除
    • del a[1] 删除指定元素
    • a.pop()可指定弹出元素，默认末尾元素
    • a.remove()根据值删除元素
  • 排序
    • a.sort()永久排序
    • a.sorted()临时排序
    • a.reverse()逆序(永久)
  • 长度 len()
  • 数字列表
    • min(a)
    • max(a)
    • sum(a)
  • 复制列表
    • b = a a改变，b也改变
    • b = a[:]a改变，b不改变
