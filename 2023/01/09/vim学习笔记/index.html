发表于 2023-01-09 更新于 2023-07-12 分类于 学习笔记
学习vim

vim

vim是一款高效的编辑器，它可以使你双手不离开键盘完成代码亦或者文档的编辑工作。

普通模式NORMAL

vim的命令可以看成<operation> <motion>

  • h j k l左下上右

    • j 5向下移动5行

  • u撤销上一步的操作 Ctrl+r恢复上一步被撤销的操作

  • i 在光标前进入编辑模式

    • I在行前进入编辑模式

  • a在光标后进入编辑模式

    • A在行前进入编辑模式

  • o在当前行的下一行插入一行进入编辑模式

    • O在当前行的上一行插入一行进入编辑模式

  • x删除当前光标下的字符

  • d 删除

    • d 3 l(->)往右删除三个字符
    • dd 剪切这一行

  • y 复制

    • y l 复制当前光标右边的一个字符
    • y 3 l 复制右边三个字符

  • p 粘贴

  • c 修改

    • c 6 删除右边6个字符并进入写入模式

    • c i w 当你的光标在单词中任意位置，删除并重新编辑此单词

    • c i "当你的光标在双引号中任意位置，删除并重新编辑双引号内的内容

  • f 行内向后查找F行内向前查找

    • d f : 一直删除到冒号位置

    • y f :一直复制到冒号

    • c f :一直删除到冒号，并进入写入模式

  • / 文件内查找

    • n查看上一次的查找内容
      • n往下切换查找选中内容
      • N往上切换查找选中内容

  • i 的in功能

  • c i "删除双引号里面的内容并且进入写入模式

  • y i "复制双引号里面的内容

  • d i "删除双引号里面的内容

  • gi 跳转到上次进行写入操作的地方并进入写入模式

写入模式

  • CTRL-H lets you remove the last character you typed (mnemonic h which in hjkl brings the cursor one space to the left)
  • CTRL-W lets you remove the last word you typed (mnemonic word)
  • CTRL-U lets you remove the last line you typed (mnemonic undo this line)

编辑模式

按:进入编辑模式

  • w 保存
  • q 退出
  • split上下分屏,Q退出
  • vsplit左右分屏,Q退出
  • tabe打开新的标签页
    • -tabnext切换到左边的标签页
    • +tabnext切换到右边的标签页

可视模式VISUAL

  • v进入可视模式

  • V(大写V)进入可视行模式

  • 在每一行前面加同样的字符my_wallpaper_

    • :
    • normal Imy_wallpaper_

  • 当选中一段内容后输入:,然后输入normal，后面输的命令可以相当于对每一行在normal模式下执行

  • ctrl+v可视块模式

  • 在文档的最左上角，ctrl+v进入可视块模式后，按G，即选中一竖行

此刻我想把下面四行wallpaper前面的都删除，并把w变成大写，在第四行进入可视块模式，选中a前面的一块

按d删除，再次进入可视块模式，输入大写I，输入W，按esc，vim自动帮我们把目标行都加了个W
